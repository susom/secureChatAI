   WHY SEPARATE EMs?

   ✅ Per-project scoping    → PID 65 gets record tools, PID 70 gets both
   ✅ Least privilege        → Bug in escalation ≠ records exposed
   ✅ Independent releases   → Ship escalation fix without touching records
   ✅ Zero-code extensible   → New tools = new EM + config.json, done
   ✅ Brain stays clean      → SecureChatAI = orchestration only
